# Microservice Application-Lab
Mit dieser Arbeit soll untersucht werden, ob sich der Ansatz einer Microservice-Architektur für den Einsatz in der Lehre eignet und was dabei zu beachten ist. Die Arbeit ist im Herbstsemester 2018 durchgeführt worden. Die Ziele des Projekts sind eine Zerlegung einer Applikation in Microservices vorzunehmen und ein Application Lab zu entwickeln. Als Grundlage für die Zerlegung wird eine Applikation aus dem Informatikmodul APPE der Hochschule Luzern verwendet.
